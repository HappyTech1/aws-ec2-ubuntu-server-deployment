# AWS EC2 Ubuntu Server Deployment and SSH Configuration

## Project Overview

This project demonstrates the deployment and configuration of an Ubuntu Server on AWS EC2. The project covers launching an EC2 instance, configuring security groups, creating SSH key pairs, and connecting securely to the server using SSH.

## Objectives

* Launch an AWS EC2 Ubuntu instance
* Configure Security Groups
* Create and use SSH Key Pairs
* Connect to the server using SSH
* Perform basic Linux administration
* Verify successful server deployment

## Tools Used

* AWS EC2
* Ubuntu Server
* SSH
* Linux Terminal

## Deployment Steps

1. Created an AWS EC2 Ubuntu instance.
2. Configured Security Group rules.
3. Generated and downloaded an SSH Key Pair.
4. Connected to the instance using SSH.
5. Updated system packages using Linux commands.
6. Verified successful server access and management.
